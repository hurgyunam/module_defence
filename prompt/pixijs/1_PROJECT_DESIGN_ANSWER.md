💻 PixiJS + TypeScript 구현 팁
이 디자인은 PixiJS의 Graphics와 Text 객체를 사용하여 매우 효율적으로 구현할 수 있습니다.

사각형 그리기 (배경 및 테두리):

new PIXI.Graphics() 객체를 생성합니다.

테두리: lineStyle(굵기, 색상 코드)를 사용하여 테두리를 설정합니다.

배경: beginFill(색상 코드)와 drawRect(x, y, width, height)를 사용하여 배경색을 채웁니다.

테두리 굵기 활용: 테두리 굵기는 유닛의 중요도나 레벨/단계를 시각적으로 강조하는 데 유용합니다.

알파벳 표시:

new PIXI.Text(알파벳, 스타일) 객체를 생성합니다.

스타일에는 글자 색상 (예: #000000), 폰트 크기 등을 지정합니다.

텍스트 객체의 위치를 사각형 타일의 중앙에 오도록 설정합니다.

코드 구조:

각 유닛 타입별로 기본적인 색상과 스타일을 정의하는 설정 파일/객체를 만들고, 이 설정을 기반으로 Graphics와 Text 객체를 생성하는 재사용 가능한 클래스 (예: SimpleUnit)를 만들면 유지보수가 쉽습니다.

```
interface UnitDesign {
    color: string;           // 배경 색상 코드 (예: '#CC0000')
    borderThickness: number; // 테두리 굵기 (예: 5)
    letter: string;          // 유닛 위에 표시될 알파벳 (예: 'T10')
}

// 예시: 레벨 10 타워 디자인
const towerLevel10Design: UnitDesign = {
    color: '#CC66FF',
    borderThickness: 5,
    letter: 'T10',
};
```
