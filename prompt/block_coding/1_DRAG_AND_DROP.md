[프레임워크 정보]
- Remix 2.17.2, React 18.3.1, Tailwind CSS

[요구사항 요약]
- 인라인 블록과 중첩(Nested) 블록의 드래그 앤 드롭 시스템.
- 자석처럼 달라붙는 스냅(Snapping) 기능과 무제한 중첩 지원.

[특별 요청]
- 프로젝트 전체에 적용하는 대신, 'app/routes/test.tsx'와 같이 단일 경로에서 독립적으로 실행 가능한 풀 코드를 작성해 줘.
- 외부 라이브러리가 필요하다면 어떤 것들을 yarn add 해야 하는지 명확히 알려줘.
- 블록들이 배치된 계층 구조를 하단에 JSON 데이터로 실시간 출력해 줘.

[핵심요소]
- Block Palette (사이드바): 드래그해서 꺼낼 수 있는 원본 블록들이 모여 있는 곳.
- Code Canvas (메인): 블록들이 자유롭게 배치되고 서로 결합되는 빈 공간.
- Visual Feedback: 자석처럼 붙을 위치를 미리 보여주는 'Drop Indicator(가이드선)' 표시.
- State Preview: 현재 캔버스에 배치된 블록들의 계층 구조(JSON)를 실시간으로 보여주는 패널. (이게 있어야 나중에 실제 데이터로 변환하기 쉽습니다.)
